Usage:
This is a sketch_pad that you can press on each button and draw different patten. 
The shape could be drawn are on the name of the button.

special Usage: 
To draw a special rectangle(square), press any key while drawing. 
To draw a special ellipses (circle), press any key while drawing. 
To draw a open polygon, start by draw a closed polygon and finish drawing by press any key. 


run the following command in terminal under the "sketch_pad" directory: 
to compile:  
	javac Main.java
to run:
	java Main
